Page to Markdown Pro

Support

Use these channels if you hit a bug, extraction issue, or listing problem.

Contact and issue reporting

  • Bug reports and feature requests: github.com/646826/page-to-md-pro/issues
  • Project homepage: 646826.github.io/page-to-md-pro/
  • Source repository: github.com/646826/page-to-md-pro

What to include in a report

  • The page URL, if it is public.
  • Whether you used page mode, selection mode, or the toolbar action.
  • A short description of the expected Markdown versus what you received.
  • Any console errors from the extension service worker or the page, if available.

Fastest path for extraction bugs

For extraction regressions, attaching the generated Markdown and a small HTML fixture usually makes the issue reproducible much faster.
